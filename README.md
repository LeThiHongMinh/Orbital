# Orbital

Step to run our application:

Run the server:
- Open a new terminal tab
- cd Orbital/server
- npm install
- npm install -g nodemon
- npm install dotenv
- npm run dev/node server.js
- Open browser: http://localhost:5000

Run the client:
- Open a new terminal tab
- cd Orbital/client
- npm install
- npm install @reduxjs/toolkit
- npm install react-redux
- npm start
